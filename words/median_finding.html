Four algorithms for median finding were written in golang, then tested on a 2015 mac book pro. The four algorithms are briefly described below and the complete implementation can be found on github.

  1. sort.Ints then take median element (default sort)
  2. quicksort then take the median element average O(nlogn) worst case O(n^2)
  3. quickselect average O(n) worst case O(n^2)
  4. determanistic quick select O(n)

The inputs where created by using the function below starting with size 2 and doubling until 2^23. 


func generate(size int) []int {
    var a []int
    for i := 0; i < size; i++ {
        a = append(a, rand.Int()/2)
    }
    return a
}

The real world run times:

While quickselect is technically an O(n^2) algorithm it has real world performance that beats deterministic quickselect with O(n) theoretical run time even for large inputs.
