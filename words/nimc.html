NIMC

NIMC(github) = Notes + vim + Client

NIMC is the renderer for your notes, instead of writing notes in some crappy WYSIWYG editor you just write it in your text editor of choice (vim) and then NIMC renders the notes.

It's programmed using custom web components and is easily extensible by creating your own custom web components to render a different file type.

Latex renderer: via Katex and chem extension for Katex

Dot render: via vis.js

Markdown render: commonmark spec via markdown-it

I plan on writing my own markdown render in order to support Katex and Vis.js in markdown and allow for even more easy extensibility.
