NIMC

NIMC(github) = Notes + Vim + Client

NIMC is the renderer for your notes. Instead of writing notes in some crappy WYSIWYG editor, just write it in a text editor of your choice (vim) and NIMC renders the notes.

It is programmed using custom web components and is easily extended by adding web components to render other file types.

Latex renderer: via Katex and chem extension for Katex

Dot renderer: via vis.js
